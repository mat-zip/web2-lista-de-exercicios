Chuck Norris Facts
Descubra coisas incríveis sobre Chuck

Crie um script para cadastrar verdades absolutas (facts) sobre Chuck Norris em um vetor/array e também apresentar uma verdade absoluta aleatória já cadastrada ao passar o mouse sobre a sua imagem. Quando o mouse é movido para fora da imagem, o texto com a verdade absoluta deve ser escondido. A apresentação e omissão do painel que contém a verdade absoluta sorteada deve ser realizada via aplicação dinâmica de estilo CSS via JavaScript. Basicamente, a página deve conter um textarea para inserir o texto do fact, um botão para acionar o evento para guardar o fact em um vetor, uma imagem de Chuck e o container para mostrar o fact sorteado.

Exemplos de Facts:

  • “Chuck Norris contou até o infinito. Foram duas vezes.”
  • “Não há o botão ‘ctrl’ no computador de Chuck, uma vez que ele sempre está sobre o controle.”
  • “Chuck destruiu a tabela periódica porque o único elemento que ele conhece é o elemento surpresa.”

Exigências

  • Ao carregar a aplicação, atribuir foco no textarea para facilitar a inserção de facts.
  • Ao clicar no botão para adicionar um item no array, apagar o conteúdo do textarea e atribuir foco ao mesmo elemento para facilitar a próxima inserção. Ao clicar, o evento de submit deve ser disparado. Ao cadastrar o fact, apresentar a mensagem de sucesso em um alert.
  • Validar o campo de textarea via atributo do HTML5 para que este seja obrigatório, ou seja, não permita o envio de espaço vazio ao clicar em adicionar.
    • Neste caso, mostrar a seguinte mensagem customizada dentro do ‘balão’ de erro renderizado pelo navegador: Chuck Norris sabe o valor de uma variável nula e ainda pode ordenar por este valor, mas ele não aceita valores vazios. Favor preencher.
  • Adicionar cada novo fact no final do array usando a função apropriada para manipulação do array.
  • Ao passar o mouse sobre a imagem, mostrar um fact aleatório. Usar o oposto deste estilo para esconder o fact quando o cursor é removido da imagem. Usar uma função randômica limitada ao tamanho do vetor para sortear o índice do fact a ser apresentado.
